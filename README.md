# my dotfiles
## tmux
- tpm install  
`git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm`
## nvim
- neovim install  
`$ brew install neovim`  
- dein install  
`$ mkdir -p ~/.cache/dein`  
`$ cd ~/.cache/dein`  
`$ curl https://raw.githubusercontent.com/Shougo/dein.vim/master/bin/installer.sh > installer.sh`  
`$ sh ./installer.sh ~/.cache/dein`
